    第一部分：核心交互与界面

    1. 两种响应模式 (全局/局部)

    • 这是什么: 此功能决定了AI角色是同时处理多个聊天，还是只专注于当前的聊天。
    • 如何操作:
      1. 当您发送消息后，可以点击屏幕最上方的灵动岛区域（显示 BLMX 的黑色条块）。
      2. 系统会弹出选项，让您选择 全局模式 或 局部模式 (非全局模式)。
        • 全局模式: AI会同时查看所有与您有新消息的聊天（包括私聊和群聊），并可能在多个聊天窗口中同时回复。适合需要多线操作、观察角色间互动的场景。
        • 局部模式: AI将只关注您当前打开的这个聊天窗口的内容，并仅针对此窗口进行回复。适合需要与特定角色进行深入、专注对话的场景。

    2. 消息未读红点提示

    • 这是什么: 当您有未读消息时，会在聊天列表对应的联系人或群聊头像上显示一个红色圆点，圆点内的数字表示未读消息的数量。
    • 如何操作:
      1. 当角色给您发消息后，聊天列表界面会自动出现红点提示。
      2. 点击进入该聊天窗口，阅读完新消息后，红点会自动消失。

    3. 置顶联系人

    • 这是什么: 您可以将重要的聊天固定在聊天列表的顶部，方便快速找到。
    • 如何操作:
      1. 在聊天列表界面，找到您想置顶的聊天（私聊或群聊）。
      2. 长按该聊天框，会弹出一个菜单。
      3. 点击菜单中的“置顶该消息”即可。再次长按并选择“取消置顶”可以恢复。
      4. 在同一个长按菜单中，您也可以选择“标为已读”，快速消除该聊天的红点提示。

    4. 切换第二页功能 (面板操作)

    • 这是什么: 在聊天输入框旁的功能面板（例如红包、转账等功能所在的位置）现在有多个页面。
    • 如何操作:
      1. 在聊天界面，点击输入框右侧的+号，打开功能面板。
      2. 您会看到面板下方有两个小圆点，代表有两个页面。
      3. 点击第二个小圆点即可切换到第二页，找到如“群红包”、“转发”等更多功能。注意：此处不能滑动切换，只能点击圆点。

    第二部分：私聊与群聊功能

    A. 通用聊天功能

    1. 转发聊天记录

    • 这是什么: 您可以将任何私聊或群聊中的一条或多条聊天记录转发给其他联系人或群聊。
    • 如何操作:
      1. 在聊天界面，点击输入框右侧+号，切换到第二页，点击“转发”功能。
      2. 聊天界面会进入“转发模式”，每条消息旁边都会出现一个复选框。
      3. 勾选您想要转发的一条或多条消息。
      4. 点击屏幕底部的“确认转发”按钮。
      5. 在弹出的联系人列表中，选择您想转发到的目标聊天，点击“确定”即可。

    2. 长截图聊天记录

    • 这是什么: 此功能可以生成一张包含当前完整聊天记录的长图片，方便保存和分享。
    • 如何操作:
      • 方式一 (功能面板): 在聊天界面，点击输入框右侧+号，切换到第二页，点击“长截图”功能。
      • 方式二 (偷看模式): 在“偷看”群聊的界面，底部会直接显示“长截图”按钮，点击即可。
      • 操作后，系统会询问您是否包含手机外框，您可以根据需要选择。截图生成后会自动下载到您的设备。

    3. 角色主动行为

    • 这是什么: 为了让世界更真实，角色们（Char）不再仅仅被动回复，他们会主动发起各种行为。
    • 具体表现:
      • 主动私聊: 角色可能会主动给您发消息。
      • 主动更改签名: 角色会根据人设和心情，自己更改他们的个性签名。您可以在他们的朋友圈主页看到。
      • 已读不回提示: 如果您给多个角色发了消息，而某个角色暂时没有回复，系统会自动提示您他“为什么已读不回”或“正在做什么”，让您了解他的状态。

    4. 专属表情包

    • 这是什么: 您可以为每个私聊和群聊设置专属的表情包库。在一个聊天中导入的表情包，不会出现在另一个聊天里。
    • 如何操作:
      1. 进入与角色A的私聊或群聊A。
      2. 点击输入框右侧的+号，然后点击“红包”图标。
        3. 请注意：此处的红包图标是进入角色/群聊专属表情包功能的入口。
      3. 在打开的面板中，点击“添加”，按照“描述URL”的格式输入表情包信息。
      4. 这样添加的表情包就成为了角色A或群聊A的专属表情包，只能在此处使用。

    5. 发送语音消息

    • 这是什么: 您可以向任何聊天发送一条模拟的语音消息。
    • 如何操作:
      1. 在聊天界面的输入框左侧，点击麦克风图标 🎤。
      2. 在弹出的输入框中，先输入语音消息想要转换成的文字内容。
      3. 在下一个弹出的输入框中，输入语音的持续秒数（一个纯数字）。

    6. 发送其他丰富消息

    • 这是什么: 您可以通过功能面板主动向角色发送转账、文件、礼物和地理位置等多种信息卡片。
    • 如何操作:
      1. 在聊天界面，点击输入框右侧的 + 号，打开功能面板。
      2. 点击对应的图标：
        • 转账: 输入金额和备注。在群聊中，您需要先选择一位接收者。
        • 礼物: 输入礼物名称和价格（可选）。在群聊中，您需要先选择一位接收者。
        • 文件: 输入您想发送的文件名（如 “学习资料.pdf”）。
        • 位置: 输入您想分享的地理位置名称。
      3. 根据提示完成操作后，相应的卡片消息就会被发送出去。

    7. 查看转发的聊天记录详情

    • 这是什么: 当您在聊天中看到一个被转发的聊天记录卡片时，可以点击它查看完整的转发内容。
    • 如何操作:
      1. 在聊天界面，找到一个标题为“聊天记录”或“转发的动态”的白色卡片消息。
      2. 直接点击该卡片。
      3. 系统会弹出一个新页面，详细展示被转发的每一条消息或朋友圈动态。

    B. 群聊专属功能

    1. 创建与偷看群聊

    • 这是什么: 您可以自由创建群聊，并且可以选择自己是否加入。如果不加入，您就能以上帝视角“偷看”群里其他角色的对话。
    • 如何操作:
      1. 在聊天列表界面，点击右上角的+号，选择“发起群聊”。
      2. 勾选您想邀请的群成员。
      3. 关键选择: 系统会询问您“是否要加入这个群聊？”
        • 选择 “是/OK”: 您将作为群主加入该群。
        • 选择 “否/Cancel”: 您不会加入该群。之后系统会要求您从被邀请的成员中选择一位作为群主。
      4. 创建成功后，对于您不在的群聊，点击进入聊天界面，底部会出现一个 “🤫 偷看他们聊天” 的按钮。点击此按钮，AI会自动生成一段群成员间的对话。

    2. 用户作为群主/管理员的操作

    • 这是什么: 当您创建群聊并加入其中时，您就是群主，拥有完整的管理权限。
    • 如何操作:
      1. 进入您是群主或管理员的群聊，点击右上角的 ... 进入群设置。
      2. 添加/踢出成员：点击成员列表最后的 + 号，可以从联系人中邀请新成员入群。点击任意群成员的头像，在弹出的菜单中可以选择“踢出群聊”。
      3. 设置/取消管理员：只有群主可以操作。点击一个非管理员成员的头像，在菜单中选择“设为管理员”。点击一个已是管理员的成员头像，可以选择“取消管理员”。
      4. 禁言/修改群昵称：点击任意群成员（群主和其他管理员除外）的头像，可以选择“禁言”并设置时长，或“修改群昵称”。
      5. 更换群头像：在群设置页面，点击“更换群聊头像”选项，输入图片URL即可。

    3. 角色作为群主/管理员

    • 这是什么: 角色（Char）现在拥有了群管理能力。如果Char是群主，他可以指定其他角色为管理员。管理员拥有部分管理权限。
    • 如何操作:
      • 任命管理员: 如果一个Char是群主，他可能会在对话中通过AI指令（SET_ADMIN）将其他成员设为管理员。您会看到“xxx已被设为管理员”的系统提示。
      • 管理员权限: 成为管理员的角色，可以对普通群成员进行禁言、修改群昵称、踢出群聊等操作。

    4. 角色主动退出群聊

    • 这是什么: 角色可能会根据剧情或人设发展，自行决定退出某个群聊。
    • 如何操作: 这是一个由AI驱动的被动功能。当条件满足时，您会在群聊中看到系统提示：“[角色名] 退出了群聊”。

    5. 群禁言功能

    • 这是什么: 群主或管理员可以禁言群内的指定成员，被禁言的成员在禁言期间无法发送消息。此功能拥有真实的计时系统。
    • 如何操作:
      • 手动禁言: 作为群主/管理员，进入群聊，点击右上角...进入群设置。点击您想操作的群成员头像，在弹出的选项中选择“禁言”，并输入禁言的分钟数。
      • 自动解禁: 到达设定的禁言时间后，系统会自动在群聊中提示“xxx的禁言已到期，自动解除”。
      • 手动解禁: 您也可以在禁言期间，再次进入群设置，点击被禁言成员的头像，选择“解除禁言”。

    6. 修改群成员昵称

    • 这是什么: 群主或管理员可以修改群内任何成员的昵称。
    • 如何操作:
      1. 作为群主/管理员，进入群聊，点击右上角...进入群设置。
      2. 点击您想修改昵称的群成员头像。
      3. 选择“修改群昵称”，并输入新的昵称。
      4. 修改成功后，群聊内会出现相应的系统提示。

    7. 修改群名与群壁纸

    • 这是什么: 您可以修改所在群聊的名称和聊天背景。
    • 如何操作:
      1. 进入群聊，点击右上角...进入群设置。
      2. 点击“群聊名称”选项，即可输入新名称。修改后群内会有系统提示。
      3. 点击“更换聊天背景”选项，输入图片URL，即可为该群聊设置专属壁纸。此操作不会有系统提示。

    8. 群聊抢红包

    • 这是什么: User和Char都可以在群聊中发红包，所有群成员（除了发送者）都会自动参与抢红包。
    • 如何操作:
      • User发红包:
        1. 在群聊界面，点击输入框右侧+号，切换到第二页，点击“群红包”图标。
        2. 输入红包名称和金额，点击发送。
        3. 发送后，系统会自动模拟其他角色抢红包的过程，并在聊天中显示谁抢了多少，以及谁是“运气王”。
      • Char发红包: 这是一个由AI驱动的行为。角色可能会在特定情境下主动发出红包。

    9. 艾特 (@) 群成员

    • 这是什么: 在群聊中快速提及某个成员，以引起其注意。
    • 如何操作: 在群聊天的消息记录中，直接点击任意一位成员的头像，他的昵称（例如@张三）就会被自动添加到您的输入框中。

    10. 解散群聊

    • 这是什么: 只有群主才能解散群聊。
    • 如何操作:
      1. 如果您是群主，进入群聊，点击右上角...进入群设置。
      2. 在页面底部，点击红色的“解散该群聊”按钮。
      3. 您可以选择保留聊天记录（群聊会被存档，可恢复）或彻底删除。

    第三部分：朋友圈功能

    1. 朋友圈可见范围

    • 这是什么: 您在发送朋友圈时，可以精确控制哪些人可以看到这条动态。
    • 如何操作:
      1. 在发布朋友圈的编辑界面，会有“谁可以看”的选项。
      2. 您可以选择“公开”、“私密”、“部分可见”（指定某些人才能看）或“不给谁看”（指定某些人不能看）。
      3. 发布后，在该条朋友圈下方，您可以点击一个小人图标，查看具体有哪些人可以看到这条动态。

    2. 转发朋友圈动态

    • 这是什么: 您可以将一条公开的朋友圈动态转发给您的某个联系人或群聊。
    • 如何操作:
      1. 在您想转发的朋友圈动态下方，找到并点击分享图标（一个评论气泡旁边的图标）。
      2. 在弹出的联系人列表中，选择您要转发的目标。
      3. 该朋友圈会以卡片的形式被发送到对应的聊天中。

    3. 私密朋友圈

    • 这是什么: 一种只有自己能看到的朋友圈，无法被点赞或评论，适合记录私人心情。
    • 如何操作:
      • User发送: 点击“我” -> “朋友圈”，进入自己的朋友圈主页。点击右上角的相机图标，在弹出的选项中选择“发表私密朋友圈”。
      • Char发送: 这是一个由AI驱动的行为，角色可能会自行发布私密朋友圈。

    4. 角色发图与朋友圈更新提示

    • 这是什么: 角色发朋友圈时可以配上图片，并且当有任何朋友圈更新时，您都会在聊天列表收到提醒。
    • 如何操作:
      • 角色发图: 这是一个由AI驱动的功能。只要角色的设定中有图片URL，AI就有概率在发朋友圈时带上图片。
      • 更新提示: 在您的聊天列表里，有一个固定的“朋友圈”聊天框。当任何角色发布了新的朋友圈，或更新了个性签名，这个聊天框就会像普通消息一样出现红点和内容提示，提醒您查看。

    5. 点赞与评论朋友圈

    • 这是什么: 您可以对他人（或其他角色）发布的朋友圈进行点赞或发表评论。
    • 如何操作:
      1. 在朋友圈信息流中，找到您想互动的那条动态。
      2. 点击动态右下角的 评论图标 💬 (两个点的气泡图标)。
      3. 在弹出的输入框中：
        • 输入 赞 或 like 然后确认，即可点赞。
        • 直接输入任何其他文字内容，即可作为评论发表。

    第四部分：用户个人化与好友管理

    1. 修改自己的昵称与头像

    • 这是什么: 您可以随时更改自己（User）在模拟器中显示的昵称和头像。
    • 如何操作:
      1. 在主界面点击底部的 “我” 标签，进入个人信息页。
      2. 修改头像：直接点击您的头像，会弹出输入框，输入新的图片URL即可更换。
      3. 修改昵称：点击您的昵称（ID下方），在弹出的输入框中输入新昵称即可。

    2. 添加好友

    • 这是什么: 添加新的AI角色作为您的联系人。
    • 如何操作:
      1. 在聊天列表界面，点击右上角的+号，选择“添加朋友”。
      2. 在弹出的输入框中，务必输入该角色的“真实姓名”。这个姓名将作为角色的唯一ID，不可与其他角色重复。
      3. 添加成功后，您可以进入与他的私聊界面，点击右上角...，再点击他的名字，即可为他设置备注，方便您识别。

    3. 修改联系人备注

    • 这是什么: 您可以为已经添加的好友设置备注名，方便识别。
    • 如何操作:
      1. 进入与该好友的私聊界面。
      2. 直接点击屏幕顶部的联系人名称。
      3. 在弹出的输入框中输入新的备注名并确认即可。

    4. 删除好友

    • 这是什么: 从您的联系人列表中移除某个角色。
    • 如何操作:
      1. 进入想删除好友的私聊页面，点击右上角的...进入联系人详情。
      2. 点击红色的“删除该联系人”按钮。
      3. 系统会提示您选择删除方式。目前版本只实现了真删除功能。
        • 真删除: 将会彻底删除该联系人以及所有与他相关的私聊记录，此操作不可逆。
        • 假删除: （待更新）未来将支持仅隐藏联系人但保留记录的删除方式。

    第五部分：其他高级功能

    1. 手动删除消息记录

    • 这是什么: 在任何聊天界面，进入一种特殊的“删除模式”，可以逐条删除您不想要的消息记录或系统提示。
    • 如何操作:
      1. 在聊天界面中，点击屏幕左上角靠近灵动岛左侧的一个隐形触发区域。
      2. 点击后，系统会提示“已进入删除模式”。此时每条消息左侧（或右侧）会出现一个 "❌" 图标。
      3. 点击您想删除的消息，确认后即可将其从聊天记录中移除。
      4. 再次点击屏幕左上角的相同位置，即可退出删除模式。

    2. 撤回自己发送的消息

    • 这是什么: 您可以撤回自己发送的消息。
    • 如何操作:
      1. 在聊天界面中，长按您自己发送的那条消息气泡。
      2. 在弹出的菜单中，选择“撤回”。
      3. 系统会提示您输入撤回的时间，您可以保持默认或自定义。确认后，该消息将被撤回，并在原位置显示“你撤回了一条消息”。

    3. 查看被撤回的消息

    • 这是什么: 当他人撤回一条消息后，您可以通过点击提示来查看被撤回的原始内容。
    • 如何操作:
      1. 当聊天中出现“‘xxx’撤回了一条消息”的灰色提示时。
      2. 点击这条灰色提示文字。
      3. 被撤回的原始内容就会在下方展开显示。

    4. 更换全局/主屏幕/设置壁纸

    • 这是什么: 除了为单个聊天设置壁纸，您还可以更换全局的聊天壁纸、App主屏幕壁纸和“设置”页面的壁纸。
    • 如何操作:
      1. 在主屏幕点击“设置”App。
      2. 选择“更换全局聊天壁纸”来更改所有未单独设置壁纸的聊天背景。
      3. 选择“更换主屏幕壁纸”来更改App列表界面的背景。
      4. 点击列表最下方的“更换设置壁纸”选项，可以为“设置”应用单独更换背景壁纸。
